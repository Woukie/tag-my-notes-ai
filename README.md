# Tag My Notes AI
- Open tagging UI from the ribbon (left toolbar) to begin tagging
- Or use the right click menu button to quickly tag the currently open note with all relevant tags
- Choose which tags, what notes and see the progress as the AI tags each note
- Tagging operations even persist through closing and re-opening of the application
- You can view, cancel and delete tagging operations from the tagging UI
- Customise the AIs train-of-thought to get more accurate results

[Warning] This is a powerful tool. Adding more reasoning steps significantly increases token cost and tagging time. For processing a large number of files, it is best to use one reasoning step. Make sure to keep {description} and {tag} placeholders when you do this.
